## HOW
1. Automator
	1. New Document -> Quick Action
	2. Add `Launch Application` from Library
	3. Select the app you want to launch
2. System settings/keyboard/ Keyboard shortcuts/...
	1. Services/ General 에 보면 내가 만든 Workflow가 있음
	2. 여기서 단축키 지정하면 됨

## 예시
- BusyCal(캘린더 앱) 열기
	-  `cmd + opt + C`
- Arc browser 열기
	-  `cmd + opt + A`
- Obsidian 열기
	-  `cmd + opt + O`
